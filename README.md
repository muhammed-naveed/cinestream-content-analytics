# CineStream Content Analytics Dashboard

Interactive Streamlit dashboard for analyzing OTT platform content performance.

## Features

- KPI Cards
- Genre Analysis
- Language Analysis
- ROI Analysis
- IMDb Analysis
- Interactive Filters
- CSV Export

## Tech Stack

- Streamlit
- Pandas
- NumPy
- Matplotlib
- Plotly

## Run

```bash
streamlit run cinestream_app.py
